# Promptimall: sammsammult lahendus

## Eesmärk

Seda promptimalli saab kasutada ülesannete puhul, kus õppimise seisukohalt on oluline näha lahenduskäiku, mitte ainult lõppvastust. Mall sobib näiteks matemaatika, algoritmide, andmebaaside ja programmeerimisülesannete jaoks.

## Promptimall

```txt
Käitu [aine/valdkonna] õppejõuna.

Ülesanne:
[lisa ülesande tekst]

Minu eesmärk:
Soovin ülesannet mõista, mitte ainult lõppvastust saada.

Palun:
1. selgita, millist meetodit kasutad;
2. lahenda ülesanne samm-sammult;
3. näita olulisi vaheetappe;
4. põhjenda, miks iga samm vajalik on;
5. anna lõpus lühike kokkuvõte ja lõppvastus;
6. kui ülesandes on mitu võimalikku lahendusviisi, maini vähemalt ühte alternatiivi.
```

## Lühem variant

```txt
Lahenda järgmine ülesanne sammsammult. Selgita kasutatud meetodit, näita olulisi vaheetappe ja anna lõpus lõppvastus.

Ülesanne:
[lisa ülesande tekst]
```

## Näide

```txt
Käitu matemaatika õppejõuna.

Ülesanne:
[lisa siia konkreetne ülesanne]

Minu eesmärk:
Soovin ülesannet mõista, mitte ainult lõppvastust saada.

Palun:
1. selgita, millist valemit või meetodit kasutad;
2. lahenda ülesanne samm-sammult;
3. näita olulisi vaheetappe;
4. põhjenda, miks iga samm vajalik on;
5. anna lõpus lühike kokkuvõte ja lõppvastus.
```

## Kasutusolukord

Seda prompti sobib kasutada siis, kui:

- ülesanne on mitmeetapiline;
- on vaja mõista lahenduse loogikat;
- soovid kontrollida oma lahenduskäiku;
- tahad näha alternatiivseid lahendusviise;
- lõppvastus üksi ei ole õppimiseks piisav.

## Soovitus

Kui vastus muutub liiga pikaks, ei tasu kohe paluda mudelil anda ainult lõppvastus. Parem on täpsustada, et vastus oleks lühike, kuid kontrollitav.

Näiteks:

```txt
Hoia vastus võimalikult lühike, kuid jäta alles peamised vaheetapid ja põhjendused.
```
